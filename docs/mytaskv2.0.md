### DAQ IDE MVP 开发任务列表（更新版）

**MVP 目标**：1.5-2 个月

**当前阶段**：集成规划

**阶段 1：Python 引擎（daq_core，优先 MQTT）**  
- 搭建 Python 项目结构  
- 实现 MQTT 客户端（订阅 topic 获取数据，发布命令）  
- 实现 MockDevice（模拟 MQTT 数据）  
- 实现核心节点函数（数学、逻辑、系统）  
- 实现基本存储组件（CSV 文件记录）  

**阶段 2：编译器（JSON → Python）**  
- 实现 .daq 文件解析器（支持 MQTT 配置）  
- 实现节点执行顺序的拓扑排序  
- 实现 Python 代码生成器（生成 MQTT 订阅/处理逻辑）  
- 使用黄金样例测试（MQTT 模拟数据）  

**阶段 3：VSCode 插件外壳**  
- 复用现有插件（王阳明）  
- 实现项目管理 + 模式切换  
- 集成 Python 进程启动（运行生成的脚本）  

**阶段 4：可视化编辑器（组件拖拽）**  
- 复用阿里低代码 + Blockly（白云）扩展连线  
- 支持拖拽组件（MQTT 设备、后端逻辑、前端控件、存储）
  比如：可以拖拽“MQTT订阅”组件（前端节点）→ 配置topic来实现和后端交互。
- 实现属性配置 + 数据绑定  

**阶段 5：仪表盘与实时显示**  
- 复用大屏（张焕青） + 控件绑定（王阳明）  
- WebSocket 实时遥测 + 历史回放  

**阶段 6：集成测试**  
- 全流程闭环：拖拽 MQTT 组件 → 配置 topic → 处理 → 显示 → 存储 CSV  
- 10 分钟上手验证  

**阶段 7：AI 演示**  
- 复用 MCP（白云）辅助组件生成  

**下一步建议**  
- 立即起步：阶段 1 先实现 MQTT 客户端 + WebSocket（paho-mqtt 订阅 topic 推送遥测）。
